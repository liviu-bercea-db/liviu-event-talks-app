# Tech Talks Event Schedule

A simple web application to display a schedule of tech talks. It includes a search feature to filter talks by category. The `index.html` file is generated by `generate_website.js`.

## Features

*   Displays a list of tech talks with titles, speakers, categories, descriptions, and times.
*   Talks are visually distinct (e.g., lunch breaks are highlighted).
*   Search functionality allows filtering talks by category (e.g., "AI", "Frontend").

## How to Use

To set up and view the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/liviu-bercea-db/liviu-event-talks-app.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd liviu-event-talks-app
    ```
3.  **Generate `index.html`:**
    This project uses `generate_website.js` to create the `index.html` file. You will need Node.js installed on your system to run this script.
    ```bash
    node generate_website.js
    ```
4.  **Open `index.html`:**
    After generating, open the `index.html` file in your preferred web browser to view the event schedule.
